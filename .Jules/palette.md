## 2025-02-15 - Radix UI Dialog Missing Title/Description
**Learning:** Shadcn UI wrapper components like `CommandMenu` use Radix UI `Dialog` internally, but often omit the mandatory `DialogTitle` and `DialogDescription` components. This triggers accessibility warnings and provides a poor screen reader experience when the dialog is opened, as context is missing.
**Action:** When implementing or modifying complex, inaccessible components like `CommandDialog` (or any Radix UI `DialogContent`), ensure `DialogTitle` and `DialogDescription` elements are added. If they shouldn't be visually visible to maintain design fidelity, use Tailwind's `sr-only` class.
