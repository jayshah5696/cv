## 2024-05-16 - Make Keyboard Shortcut Hints OS-Aware
**Learning:** Hardcoding '⌘' (Command) for keyboard shortcuts in UI hints confuses Windows/Linux users, making them guess the correct modifier key (Ctrl vs Alt). Screen readers also don't always accurately announce '⌘' as "Command".
**Action:** When displaying keyboard shortcuts in the UI, detect the user's OS on the client-side to conditionally render '⌘' for Mac and 'Ctrl' for others. Always wrap the shortcut text in a `<kbd>` element with a descriptive `aria-label` like "Command J" or "Control J" so screen readers explicitly read out the keys rather than skipping symbolic characters.
